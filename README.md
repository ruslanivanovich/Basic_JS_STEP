# Fitness First – Trainers Catalog

This project is a trainers catalog web application built using **JavaScript (Vanilla JS)**, HTML and CSS.

The application allows users to browse fitness trainers, filter them by different criteria and view additional information through modal windows.

## 🚀 Technologies

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ⚙️ Features

* Trainers catalog
* Filtering by category and direction
* Sorting by name and experience
* Modal windows with trainer details
* Dynamic rendering of trainers using JavaScript
